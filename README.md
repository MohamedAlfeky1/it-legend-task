# IT Legend Course Page

A responsive Next.js course details page for an online learning experience. It includes a video player with theater mode, course materials, curriculum cards, comments, an ask-question modal, a leaderboard, and a mock exam flow.

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [https://it-legend-task-lilac.vercel.app/](https://it-legend-task-lilac.vercel.app/) to view the app.

## Scripts
